1.5.6  2026-07-18
  * Chart timezone fix, properly this time. 1.5.5 shifted timestamps to the
    station's wall-clock but ECharts still positioned them in the viewer's
    browser zone, so charts were still offset abroad (and could double-offset).
    The charts now render with useUTC, so ECharts treats the shifted timestamps
    as-is with no browser offset. Chart times read in the station timezone
    identically for every visitor, matching "Latest update". Set chart_tz.

1.5.5  2026-07-18
  * Fixed chart times being wrong for visitors in a different timezone than the
    station. ECharts positions points and gridlines in the viewer's browser
    zone, so relabeling ticks with chart_tz was not enough: a visitor abroad saw
    the plot shifted by the station-vs-viewer offset (e.g. one hour off in
    Switzerland for a Greek station). Timestamps are now shifted into the
    station's wall-clock before plotting, so both point positions and labels
    read in the station's timezone for every visitor, DST included. Requires
    chart_tz set (or a detectable server timezone).

1.5.4  2026-07-17
  * New Extras.chart_tz: set the station's IANA timezone (e.g. Europe/Athens)

1.5.0  2026-07-16
  * HF Propagation card (optional): solar flux, sunspots, A-index and K-index
    plus a colour-coded HF band-conditions table (day/night, Good/Fair/Poor)
    for amateur radio. Data is fetched server-side from HamQSL (N0NBH) once per

    with solar = true in [Extras]. Translated in all seven languages.

1.0.0  2026-07-02
  * First public release.
  * Interactive per-metric charts (Apache ECharts, self-hosted): temperature
    and derived, humidity, barometer, wind speed and gust, wind direction,
    wind vector, wind rose, rain, rain rate, UV, solar radiation,
    evapotranspiration, cloud base. Each on its own card. Current charts show
    the last 24 hours; period pages show their calendar span.
  * Multi-period pages: Current, Yesterday, Week, Month, Year, Last Year,
    Archive, About. Yesterday is the true midnight-to-midnight day.
  * Current-conditions hero (temperature, feels-like, today's high/low), with
    today's high/low and time shown beside each current value.
  * Sun and Moon card with a rendered moon phase.
  * Rain totals (today/month/year) and today's ET in the values.
  * Auto-discovered extra sensors (soil, leaf, air quality, lightning, battery).
  * About page: station facts read automatically plus editable prose/contact,
    optional about.inc override, coordinate privacy toggle, hardware override.
  * Multi-language with an in-page switcher: English, Greek, Spanish, French,
    German, Italian, Portuguese. The browser language is auto-selected and the
    choice remembered; labels and charts swap live. Add a language with one
    lang/*.conf file.
  * Follows your WeeWX unit system (us/metric/metricwx) and timezone; the skin
    does not force units. Chart times are shown in the station's timezone.
  * Config-driven theming: modern and classic layouts, light/dark/auto mode,
    colors, fonts, density, all from weewx.conf. No template editing.
  * Auto-refresh, amateur-station disclaimer, footer version, WX favicon.
  * NOAA monthly and yearly reports.
